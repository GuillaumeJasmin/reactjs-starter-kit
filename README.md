ReactJS Starter Kit
===================

* React 16
* React Router 4
* Webpack 4
* ESLint with Airbnb rules
* CSS Modules
* PostCSS
  * autoprefixer
  * nested css
  * vars from `config.json`
* Unit test with jest

# Build

### install dependencies
```npm install```

### start local (watch)
```yarn start```

### build dev
```npm run build:dev```

### build prod
```npm run build```

### run test
```npm run test```

### remove build folder
```run run clean```
